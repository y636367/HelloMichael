마이클과 그가 데려온 유령들입니다. 생존을 위해 유령들을 놀아주세요. <br>
(위 유령들을 피해 생존하세요.)<br>

- #### 마이클은 시도때도 없이 나타나 당신을 괴롭힐 것입니다. 만일 당신이 먼저 발견했다면 그를 피해 움직이세요.
- #### 기는자는 맵 곳곳을 맴돌 것입니다. 녀석이 보인다면 움직이지 않는 것을 추천드립니다.(소리를 내지 않고 움직인다면...?)
- #### 호기심에 못이겨 상자에 다가간다면 녀석은 당신을 상당히 고통스럽게 할 것입니다. (충분한 버틸만한 체력이 있다면 녀석을 지나치게 가능할지도...?)
